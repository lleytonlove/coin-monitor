# coin-monitor

## Installation

1. Download the installer for your operating system:

> **Important**  
> Linux is not currently supported.

- [![Windows](https://img.shields.io/badge/Windows-Installer-blue?logo=windows)](https://github.com/lleytonlove/coin-monitor/releases/download/v0.0.1/coin_0.1.0_x64_en-US.msi)

2. Run the installer and follow the prompts.  
3. Launch `app` from your applications menu.
